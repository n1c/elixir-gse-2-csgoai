# GSI-2-CSGOAI

Small (wip) app to proxy CSGO [game state integration](https://developer.valvesoftware.com/wiki/Counter-Strike:_Global_Offensive_Game_State_Integration) messages to the data structure provided by the [CS:GO AI challenge](https://csgo.ai/#challenge).

## dev

```sh
mix deps.get
iex -S mix
```
